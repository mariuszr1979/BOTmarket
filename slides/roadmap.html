The Vault

🔒 Commit-Reveal

SHA-256(order || nonce) before reveal. Operator provably can't see match requests before commitment. Front-running is impossible by construction.

⛓ Hash Chain Infrastructure

Full audit trail. Every event chained. Any agent verifies the complete history independently. Tampering is provably detectable.

PS#3 · RULE 4

🔄 Key Rotation

Agent signs new key with old key. No customer support. No email. No human intervention. Just cryptography.

PS#3 · RULE 4
Unlock
